# Lomhea

Lomhea is a community-powered travel and discovery platform for Cambodia. It helps people explore verified places across the country, from well-known landmarks to lesser-known local spots, through an interactive map, user submissions, reviews, and curated featured destinations.

The project was built to feel more like a living map than a static travel guide. Users can discover places, save favorites, leave reviews, request new locations, and receive updates when their submissions are approved. On the admin side, moderators can review incoming requests, manage places and reports, monitor user activity, and keep the platform trustworthy as it grows.

## What it does

- Interactive map-based exploration for destinations across Cambodia
- Community place submissions with admin approval workflows
- User accounts with email/password and social login support
- Favorites, reviews, and profile management
- Admin dashboard for places, users, reports, requests, and security
- Image uploads for user-submitted content
- Push notifications and real-time update flows
- Bakong KHQR payment support for featured place promotion

## Tech stack

- Next.js 16 with the App Router
- React 19 and TypeScript
- Prisma with PostgreSQL
- Better Auth for authentication
- Zustand and TanStack Query for client state and server data
- MapLibre and React Map GL for map experiences
- Cloudinary for media uploads
- Redis and Pusher for caching and real-time features

## Why this project stands out

This is not just a landing page or CRUD demo. Lomhea combines product design, authentication, database modeling, admin tooling, user-generated content moderation, cloud media handling, notifications, and payments in one full-stack application. It reflects the kind of real-world tradeoffs involved in building a platform that people can actually use.
